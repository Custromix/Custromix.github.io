Northmarina

Site de gestion

Northmarina est un de mes tout premier site dynamique. Le but de ce site est la gestion d'employé pour une entreprise fictive sur le jeu-vidéo Grand Theft Auto 5 avec le mod FiveM. En effet j'ai voulu mélanger mes heures de farm intensive de ce jeu avec ce que je fais en BTS.

C'est un site en php from scratch utilisant une base de données MySql. La template s'appelle "SB-ADMIN 1" qui est une template gratuite avec bootstrap.
